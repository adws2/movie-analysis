## Motivation
영화에서 입소문의 효과가 클까?  
영화 누적관람객을 전염병 모델인 SI model 에 피팅시킨다.

## Conclusion
- 전염병의 효과는 관람객 수의 로지스틱 함수, 광고의 효과는 관람객 수의 선형함수로 생각할 수 있다. 따라서 특정 영화의 전파는 입소문보다 광고의 효과가 크다고 생각해 볼 수 있다.
